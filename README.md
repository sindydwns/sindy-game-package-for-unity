# Sindy

Sindy Game Package.

## Installation

이 패키지는 R3를 강력하게 의존합니다.
- https://github.com/Cysharp/R3

R3를 설치하기 위한 방법
- Package Manager에서 "Add package from git URL..."을 선택하여 "https://github.com/GlitchEnzo/NuGetForUnity.git?path=/src/NuGetForUnity"를 입력하여 NuGetForUnity 패키지를 설치합니다.
- NuGetForUnity 패키지를 설치한 후, "NuGet" 메뉴에서 "Manage NuGet Packages"를 선택하여 "R3" 패키지를 검색하고 설치합니다.
- NuGetForUnity를 사용하여 R3 패키지를 설치한 후, Unity의 Package Manager에서 "Add package from git URL..."을 선택하여 "https://github.com/Cysharp/R3.git?path=src/R3.Unity/Assets/R3.Unity"를 입력하여 R3.Unity 패키지를 설치합니다.
